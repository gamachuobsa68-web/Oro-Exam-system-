# 🎓 Oro Exam System

Mobile-based Kivy Exam System for students and teachers.

---

## ✨ Features
- Student login
- MCQ exam system
- Auto grading
- Leaderboard
- PDF result export
- SQLite database

---

## 🛠 Requirements
- Python 3.10+
- Kivy
- Buildozer (for APK)

---

## 🚀 How to Run

```bash
pip install kivy
python main.py
